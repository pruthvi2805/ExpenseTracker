# Expense Tracker

A minimal, privacy-first expense tracker that runs entirely in your browser. No backend, no accounts — your data stays on your device.

## Features

- Track expenses by category
- Set monthly budgets
- See spending vs. budget at a glance
- Works offline (localStorage)
- Mobile-friendly

## Live Site

https://expense.kpruthvi.com

## Tech Stack

- Vanilla JavaScript (ES6+)
- Tailwind CSS (via CDN)
- Vite (build tool)
- localStorage for data persistence

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Data & Privacy

All data is stored in your browser's localStorage. Nothing is sent to any server. Use responsibly — clearing browser data will erase your expenses.
